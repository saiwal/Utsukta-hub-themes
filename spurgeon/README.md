### Spurgeon 

This theme is based on the template ['Spurgeon' by Styleshout](https://styleshout.com/free-templates/spurgeon/) which is a clean and minimal template.

#### Features

##### Channel 'Hero' widget
The channel page has a *hero* widget to showcase featured posts. To include post(s) in this section, simply goto display settings and under 'custom theme settings' specify the category of posts to be used to display as a hero slider. Any post within this category will be showcased in this section.
