# Addons (प्लगइन)

Addons section plugins install, enable, disable और remove करने देता है।

[IMAGE: Addons section installed और available plugin lists के साथ]

## Installed Plugins

Enabled plugin active है — उसकी features सभी users को उपलब्ध हैं। Disabled plugin installed है लेकिन चल नहीं रहा।

[IMAGE: Installed plugin card name, version, description और action buttons के साथ]

## Available Plugins

Install नहीं हुए plugins नीचे दिखते हैं। **Install** क्लिक करें।

## Plugin Install करना

1. Available list में plugin खोजें।
2. **Install** क्लिक करें।
3. Plugin तुरंत activate।

[IMAGE: Available plugin list Install button के साथ]

## Plugin Disable करना

**Disable** क्लिक करें। Plugin बंद लेकिन data और configuration बनी रहती है। **Enable** से फिर शुरू।

**Disable करें जब:** testing करना हो कि कोई plugin problem cause कर रहा है या नहीं।

## Plugin Uninstall करना

**Uninstall** — code completely remove। कुछ plugins अपना data भी delete करते हैं।

> ⚠ Uninstall करने पर plugin-specific user data delete हो सकता है। Live hub पर uninstall से पहले plugin documentation देखें।

## Active Plugins देखना

**Summary** page पर भी active plugins badge list के रूप में दिखते हैं।
