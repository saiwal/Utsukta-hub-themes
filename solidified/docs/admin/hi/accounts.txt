# Accounts (खाते)

Accounts section आपके hub पर हर registered account की सूची दिखाता है।

[IMAGE: Accounts table email, channels, created date, status और action columns के साथ]

## Accounts Table

| Column | विवरण |
|--------|-------|
| **Email** | Account का registration email |
| **Channels** | इस account के कितने channels हैं |
| **Created** | Account कब register हुआ |
| **Status** | Blank = active; **Blocked** badge = account blocked है |
| **Actions** | Block / Unblock / Delete बटन |

## Account Block करना

Account row पर **Block** क्लिक करें। Account तुरंत suspend — user login नहीं कर सकता।

Blocked account में Status column में लाल **Blocked** badge दिखता है। Access वापस देने के लिए **Unblock** क्लिक करें।

**Block करें जब:** user नियमों का उल्लंघन करे लेकिन आप बाद में unblock करने या content review की संभावना रखना चाहते हों।

[IMAGE: Blocked account row Blocked badge और Unblock button के साथ]

## Account Delete करना

**Delete** क्लिक करें। Confirmation dialog:

> **इस account को delete करें? यह वापस नहीं होगा।**

Confirm करने पर account, उसके सभी channels, पोस्ट और सारा associated data permanently हट जाता है।

**Delete करें:** spam accounts, clearly illegitimate accounts, या user के खुद के request पर।

> ⚠ Deletion अपरिवर्तनीय है। अनिश्चित हों तो पहले Block करें।

अगर registration policy "with approval" है, तो नए account applications **Summary** में "Pending registrations" के अंतर्गत दिखते हैं।
