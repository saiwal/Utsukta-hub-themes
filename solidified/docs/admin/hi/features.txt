# Features (सुविधाएं)

Features section optional site features enable या disable करने देता है। बदलाव आपके hub के सभी users को affect करते हैं।

[IMAGE: Features section grouped toggles के साथ]

## कैसे काम करता है

Features **sections** में grouped हैं (जैसे "Post and Content", "Profile", "Network")। हर feature एक toggle है।

- **Checked** = feature सभी users के लिए enabled
- **Unchecked** = feature hub-wide disabled

**Save** क्लिक करें।

[IMAGE: Feature toggle rows labels और checkboxes के साथ]

## सामान्य Feature Groups

### Post and Content
Posts लिखते समय users क्या कर सकते हैं — spoiler tags, content warnings, custom permissions, formatting options।

### Profile
Profile fields और features — birthday display, keywords, custom profile fields।

### Network / Stream
Stream-related features — advanced sorting, "following" filter, specific view modes।

### Connection / Privacy
Privacy-related features — object capabilities, group-actor channels, advanced permission interfaces।

## Impact

Feature बदलाव save के बाद तुरंत लागू होते हैं। Users को reload की ज़रूरत नहीं।

> **सुझाव:** Features incrementally enable करें। एक साथ सब चालू करने से users confuse हो सकते हैं।

## Save करना

ऊपर दाईं ओर **Save** क्लिक करें। Success message confirm करता है। Error होने पर कोई बदलाव नहीं होता।
