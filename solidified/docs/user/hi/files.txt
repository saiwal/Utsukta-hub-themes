# फ़ाइलें (क्लाउड स्टोरेज)

Files ऐप (`/files` या चैनल के Cloud टैब से) Hubzilla के अंदर व्यक्तिगत क्लाउड फ़ाइल स्टोरेज देता है। फ़ाइलें upload, व्यवस्थित और शेयर की जा सकती हैं।

[IMAGE: Files पेज फ़ोल्डर/फ़ाइल सूची के साथ]

## फ़ाइलें ब्राउज़ करना

मुख्य Files व्यू में फ़ाइलें और फ़ोल्डर सूची में दिखते हैं। हर पंक्ति में:
- फ़ाइल/फ़ोल्डर आइकन
- नाम, आकार, अंतिम संशोधन तारीख

फ़ोल्डर में जाने के लिए उस पर क्लिक करें। फ़ाइल preview या download करने के लिए उस पर क्लिक करें।

[IMAGE: फ़ाइल सूची फ़ोल्डर और फ़ाइल rows के साथ]

## फ़ाइलें upload करना

**Upload** बटन से अपने device से फ़ाइलें जोड़ें। एक साथ कई फ़ाइलें upload की जा सकती हैं।

[IMAGE: Upload बटन और file picker]

## फ़ोल्डर

- **New folder** क्लिक करें।
- फ़ाइलें move action से व्यवस्थित करें।

## फ़ाइलें शेयर करना

हर फ़ाइल का एक shareable link होता है। link की दृश्यता फ़ाइल की प्राइवेसी सेटिंग पर निर्भर है:
- **Public** — link वाला कोई भी
- **Connections only** — केवल आपके connections
- **Private** — केवल आप

## किसी और की Files देखना

`/files/<उनका-nick>` या उनके चैनल के Cloud टैब पर जाएं।

> **ध्यान दें:** Cloud स्टोरेज hub admin द्वारा enable किया जाता है। अगर Files टैब नहीं दिख रहा, तो यह feature आपके hub पर उपलब्ध नहीं हो सकता।
