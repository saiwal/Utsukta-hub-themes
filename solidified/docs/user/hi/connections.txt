# कनेक्शन, डायरेक्टरी और प्राइवेसी ग्रुप

Connections section (`/directory`) से connections manage करें, नए लोग खोजें और groups बनाएं।

[IMAGE: Connections shell view sections tabs के साथ]

## आपके Connections

**Connections** टैब आपके सभी connections की सूची:
- Avatar, display name, channel address
- Connection role
- Actions: profile देखें, role edit करें, remove करें

[IMAGE: Connections सूची avatars, नाम और action बटन के साथ]

## Directory

**Directory** टैब से इस hub और federation के channels खोजें:
- नाम या keyword से search करें
- Public profiles का card grid ब्राउज़ करें
- card क्लिक करने पर mini-profile modal में Connect बटन दिखता है

[IMAGE: Directory card grid profile cards के साथ]

[IMAGE: Directory entry modal Connect बटन के साथ]

## Connected Hubs

**Hubs** टैब (अगर उपलब्ध हो) federated hubs दिखाता है।

## Privacy Groups

Privacy Groups connections को named lists में व्यवस्थित करते हैं। किसी group को target करके post करें — केवल उस group के सदस्य देखेंगे।

### Group बनाना

1. **Privacy Groups** टैब पर जाएं।
2. **New group** क्लिक करें।
3. Group को नाम दें।
4. Connections के नाम क्लिक करके group में जोड़ें।

[IMAGE: Privacy groups सूची]

[IMAGE: Privacy group detail view member list के साथ]

### Post करते समय Groups उपयोग करना

Post composer में privacy को **Custom** करें और एक या अधिक groups चुनें। केवल चुने हुए groups और व्यक्ति post देखेंगे।

## Connection Request भेजना

किसी के channel page पर जाएं और **Follow** या **Connect** क्लिक करें। Accept होने पर दोनों की streams में एक-दूसरे की posts दिखेंगी।

## Affinity

हर connection का **affinity score** (0–100) होता है। उच्च score = "करीबी" connection। Network के affinity filter से high-affinity connections की posts ही देखें।

Connection edit view में affinity सेट करें।
