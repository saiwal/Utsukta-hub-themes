# पोस्ट लिखना

HQ डैशबोर्ड या अपने चैनल वॉल से पोस्ट लिख सकते हैं। कम्पोज़र दोनों जगह एक जैसा काम करता है।

[IMAGE: पूरा पोस्ट कम्पोज़र टूलबार, टेक्स्ट एरिया और प्राइवेसी पिकर के साथ]

## पोस्ट लिखना

टेक्स्ट एरिया क्लिक करें — पूरा कम्पोज़र खुल जाएगा। टेक्स्ट एरिया में पोस्ट टाइप करें। पोस्ट **BBCode** फ़ॉर्मेटिंग सपोर्ट करता है — टूलबार टैग डाल देती है या आप खुद टाइप कर सकते हैं।

### फ़ॉर्मेटिंग टूलबार

| बटन | काम | BBCode |
|-----|-----|--------|
| **B** | Bold | `[b]text[/b]` |
| *I* | Italic | `[i]text[/i]` |
| U | Underline | `[u]text[/u]` |
| S | Strikethrough | `[s]text[/s]` |
| `< >` | Code | `[code]…[/code]` |
| 🔗 | लिंक डालें | `[url=…]label[/url]` |
| 🖼 | इमेज URL | `[img]…[/img]` |
| Spoiler | छुपाने योग्य ब्लॉक | `[spoiler]…[/spoiler]` |

पहले टेक्स्ट चुनें, फिर बटन क्लिक करें — चुना हुआ टेक्स्ट टैग में रैप हो जाता है।

[IMAGE: कम्पोज़र टूलबार बटन लेबल के साथ]

## किसी को mention करना

`@` के बाद नाम टाइप करें। मेल खाने वाले नामों का पॉपअप आता है — उसमें से चुनें।

[IMAGE: Mention पॉपअप contact suggestions के साथ]

## प्राइवेसी कंट्रोल

**प्राइवेसी पिकर** (टेक्स्ट एरिया के नीचे) यह तय करता है कि पोस्ट कौन देखेगा:

| सेटिंग | कौन देखेगा |
|--------|-----------|
| **Connections** | आपके सभी कनेक्शन |
| **Public** | सभी — विज़िटर और पब्लिक स्ट्रीम भी |
| **Custom** | चुनिंदा कनेक्शन या प्राइवेसी ग्रुप |
| **Myself** | केवल आप |

[IMAGE: प्राइवेसी पिकर ड्रॉपडाउन विकल्पों के साथ]

## Submit करना

**Post** बटन क्लिक करें (या Display Settings में सेट शॉर्टकट)। ड्राफ्ट टाइप करते समय अपने आप सेव होता रहता है।

## ड्राफ्ट

कम्पोज़र बंद करने पर कंटेंट ड्राफ्ट के रूप में सेव होता है। HQ के **Drafts विजेट** से वापस जाएं।

## पोस्ट पर प्रतिक्रिया

स्ट्रीम में हर पोस्ट के नीचे एक्शन बटन होते हैं:

[IMAGE: पोस्ट एक्शन बार रिएक्शन बटन के साथ]

| एक्शन | विवरण |
|-------|-------|
| 👍 Like | सहमति ज़ाहिर करें। दोबारा क्लिक करने पर हटा दें। |
| 👎 Dislike | असहमति। दोबारा क्लिक करने पर हटाएं। |
| 🔁 Repeat | पोस्ट को अपने followers में reshare करें। |
| 💬 Comment | कमेंट थ्रेड खोलें और जवाब लिखें। |
| ⭐ Star | पोस्ट बुकमार्क करें। Starred फ़िल्टर में दिखेगी। |
| 🗑 Delete | पोस्ट हटाएं (केवल अपनी पोस्ट पर)। |

Like और Dislike count बटन के बगल में दिखता है। Count पर क्लिक करने से देख सकते हैं किसने react किया।

## टिप्पणी करना

💬 क्लिक करें — पोस्ट के नीचे कमेंट थ्रेड खुलता है। नीचे input में जवाब लिखें और send बटन दबाएं।

[IMAGE: कमेंट थ्रेड खुला हुआ reply input के साथ]

## पोस्ट edit करना

अपनी पोस्ट पर **⋯ मेनू** या **edit बटन** से body या title बदलें। बदलाव federated होते हैं।

## Reshare करना

**Repeat** क्लिक करें। ऊपर अपना comment जोड़ सकते हैं।

## RSVP (Events)

इवेंट पोस्ट पर Like/Dislike की जगह:
- **Accept** — शामिल होऊंगा
- **Decline** — नहीं आ सकता
- **Maybe** — शायद

ये mutual exclusive हैं — एक चुनने पर बाकी हट जाते हैं।
