# कैलेंडर

Calendar ऐप (`/calendar/<nick>`) से इवेंट बनाएं और manage करें। इवेंट connections के साथ शेयर या पब्लिक बनाए जा सकते हैं।

[IMAGE: Calendar पेज monthly view events के साथ]

## इवेंट देखना

Calendar मासिक ग्रिड में इवेंट दिखाता है। इवेंट पर क्लिक करें — पूरी details।

- previous/next तीर से महीने बदलें।

[IMAGE: मासिक calendar ग्रिड dates पर event chips के साथ]

## इवेंट बनाना

**+ New event** बटन या grid पर किसी तारीख पर क्लिक करें।

| Field | विवरण |
|-------|-------|
| **Title / Summary** | इवेंट का नाम |
| **Start date/time** | कब शुरू होगा |
| **End date/time** | कब खत्म होगा (वैकल्पिक) |
| **All day** | पूरे दिन का इवेंट |
| **Location** | कहाँ (वैकल्पिक) |
| **Description** | विवरण |

[IMAGE: Event creator modal सभी fields के साथ]

**Save** क्लिक करें — इवेंट calendar पर दिखेगा और stream में पोस्ट होगा।

## RSVP

Stream में event posts पर **Accept**, **Decline**, और **Maybe** बटन। आपका जवाब event creator को जाता है।

[IMAGE: Stream में event post RSVP बटन और response count के साथ]

## Upcoming Events विजेट

HQ पर **Upcoming Events विजेट** अगले 30 दिनों के events दिखाता है। उस विजेट के **+** बटन से HQ छोड़े बिना event बनाएं।

[IMAGE: HQ पर Upcoming events विजेट]

## किसी और का Calendar

`/calendar/<उनका-nick>` पर जाएं — उनके पब्लिक events देखें और RSVP करें।

> **ध्यान दें:** Calendar ऐप चैनल पर install होना चाहिए।
