# नोटिफ़िकेशन

नोटिफ़िकेशन विजेट अधिकांश पेज पर **राइट साइडबार** में रहता है। यह आपकी पोस्ट पर activity, नए connections, संदेश और आगामी इवेंट की जानकारी देता है।

[IMAGE: राइट साइडबार में नोटिफ़िकेशन विजेट आइटम सूची के साथ]

## नोटिफ़िकेशन के प्रकार

| आइकन | घटना |
|------|------|
| 💬 | किसी ने आपकी पोस्ट पर टिप्पणी की |
| 👍 | किसी ने आपकी पोस्ट like की |
| 🔁 | किसी ने आपकी पोस्ट repeat की |
| 👤 | नई connection request मिली |
| ✅ | Connection request accept हुई |
| ✉ | नया private message |
| 📅 | आगामी इवेंट / आज का इवेंट |
| 🎂 | आगामी जन्मदिन |
| ⚙ | System नोटिफ़िकेशन |

## नोटिफ़िकेशन पढ़ना

- **अपठित** नोटिफ़िकेशन हाइलाइट होते हैं।
- नोटिफ़िकेशन क्लिक करें — सीधे उस पोस्ट, संदेश या इवेंट पर पहुँचें।
- विजेट के ऊपर **Mark all read** (या tick आइकन) क्लिक करें — सभी अपठित एक साथ clear हो जाते हैं।

[IMAGE: नोटिफ़िकेशन विजेट अपठित आइटम और "Mark all read" बटन के साथ]

## नोटिफ़िकेशन बैज

अपठित नोटिफ़िकेशन होने पर टॉप बार या बॉटम टैब बार के नोटिफ़िकेशन बेल आइकन पर count बैज दिखता है।

[IMAGE: Nav बार पर अपठित नोटिफ़िकेशन बैज]

## नोटिफ़िकेशन कॉन्फ़िगर करना

**Settings → Notifications** पर जाएं — चुनें कि कौन सी घटनाएं:
- **Email नोटिफ़िकेशन** trigger करें (आपके email पर)
- **Visual नोटिफ़िकेशन** दिखाएं (विजेट और बैज में)

पूरी सूची के लिए [settings.txt](settings.txt) देखें।
