# नोट्स

Notes आपके चैनल से जुड़ा एक निजी scratchpad है। Notes कभी भी streams में नहीं दिखते, federate नहीं होते, और केवल आपको दिखते हैं।

`/notes` पर जाएं या sidebar से **Notes** खोलें।

## नोट लिखना

पेज के ऊपर composer में टाइप करें और **Save** क्लिक करें। Notes BBCode, Markdown, या plain HTML को सपोर्ट करते हैं — save बटन के बगल में dropdown से format चुनें।

माउस तक पहुँचे बिना save करने के लिए **Ctrl+Enter** दबाएं।

## नोट Edit करना

**Edit** बटन देखने के लिए note card पर hover करें। Composer मौजूदा content के साथ load होता है — update के लिए save करें या discard के लिए **Cancel** करें।

## नोट Delete करना

Note card पर hover करें और **Delete** क्लिक करें। एक inline confirmation prompt दिखेगा — confirm करने के लिए फिर से **Delete** क्लिक करें, या वापस जाने के लिए **Cancel**।

## Format समर्थन

| Format | विवरण |
|--------|-------|
| BBCode | Default। `[b]`, `[i]`, `[url]` आदि के साथ rich text। |
| Markdown | Headings, bold, italics, links, code blocks। |
| HTML | Raw HTML। Display से पहले sanitise होता है। |

> Notes व्यक्तिगत और निजी हैं — ये आपके सर्वर या नेटवर्क पर किसी और के साथ share नहीं होते।
