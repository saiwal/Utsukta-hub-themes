# वेबपेज

Webpages ऐप आपके चैनल पर static pages बनाने देता है — व्यक्तिगत वेबसाइट या landing page जैसा। हर page का अपना URL होता है।

`/webpages/<nick>` पर जाएं।

[IMAGE: Webpages लिस्टिंग page titles के साथ]

## Pages सूची

मुख्य Webpages view सभी published pages दिखाता है। page शीर्षक क्लिक करके पढ़ें।

[IMAGE: Webpages सूची page titles और descriptions के साथ]

## Page पढ़ना

Pages formatted HTML के रूप में render होते हैं — text, images, links और embedded content हो सकते हैं।

[IMAGE: एकल webpage formatted content के साथ]

## Page बनाना

चैनल owner के रूप में **New page** क्लिक करें:
- **Title** — page heading और listing में दिखता है
- **URL name** — page के पते में उपयोग होने वाला slug
- **Content** — BBCode
- **Privacy setting**

[IMAGE: Webpage editor title, slug और content fields के साथ]

## Edit और Delete

अपने pages पर page view या listing से Edit और Delete उपलब्ध हैं।

> **ध्यान दें:** Webpages ऐप चैनल पर install होना चाहिए।
