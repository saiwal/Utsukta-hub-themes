# Notes

Notes is a private scratchpad attached to your channel. Notes never appear in streams, are never federated, and are only visible to you.

Navigate to `/notes` or open **Notes** from the sidebar.

## Writing a Note

Type in the composer at the top of the page and click **Save**. Notes support BBCode, Markdown, or plain HTML — choose the format from the dropdown next to the save button.

Press **Ctrl+Enter** to save without reaching for the mouse.

## Editing a Note

Hover a note card to reveal the **Edit** button. The composer loads with the existing content; save to update or **Cancel** to discard.

## Deleting a Note

Hover a note card and click **Delete**. A confirmation prompt appears inline — click **Delete** again to confirm, or **Cancel** to go back.

## Format Support

| Format | Notes |
|--------|-------|
| BBCode | Default. Rich text with `[b]`, `[i]`, `[url]`, etc. |
| Markdown | Headings, bold, italics, links, code blocks. |
| HTML | Raw HTML. Sanitised before display. |

> Notes are personal and private — they are not shared with anyone else on your server or the wider network.
