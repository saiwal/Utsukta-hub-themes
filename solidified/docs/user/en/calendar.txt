# Calendar

The Calendar app (`/calendar/<nick>`) lets you create and manage events. Events can be shared with connections or made public and appear in the network stream as event posts.

[IMAGE: Calendar page showing a month view with events]

## Viewing Events

The calendar displays events in a monthly grid. Each event shows its title on the day it starts.

- Click an event to see its full details (title, time, location, description).
- Navigate between months with the previous/next arrows.

[IMAGE: Monthly calendar grid with event chips on dates]

## Creating an Event

Click the **+ New event** button or click directly on a date in the grid.

The event creator form has:

| Field | Description |
|-------|-------------|
| **Title / Summary** | The event name |
| **Start date/time** | When the event begins |
| **End date/time** | When it ends (optional) |
| **All day** | Toggle to mark as an all-day event |
| **Location** | Where it takes place (optional) |
| **Description** | More details about the event |

[IMAGE: Event creator modal with all fields]

Click **Save** to publish the event. It will appear on your calendar and be posted to your stream so connections can see it.

## RSVP

Event posts in the stream have **Accept**, **Decline**, and **Maybe** buttons. Your response is shared back to the event creator. The event post shows a count of responses.

[IMAGE: Event post in stream with RSVP buttons and response counts]

## Upcoming Events Widget

On your HQ dashboard, the **Upcoming Events widget** shows the next 30 days of your events at a glance. Click the **+** button on that widget to quickly create an event without leaving HQ.

[IMAGE: Upcoming events widget on HQ]

## Visiting Someone Else's Calendar

Navigate to `/calendar/<their-nick>` to see public events on another channel. You can RSVP to those events too.

> **Note:** The Calendar app must be installed on the channel.

## CalDAV Calendars Widget

Shown in the sidebar on your own Calendar page. Lists your CalDAV calendars, lets you create new ones, toggle which are visible, and share or unshare a calendar with your connections.

## Event Card Widget

An opt-in showcase card that highlights one specific upcoming event you choose — its title, date/time, and location, with a link to the full calendar. Add it from the widget picker, then use its gear icon to pick the event; add several copies to feature more than one.
