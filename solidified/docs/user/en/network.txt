# Network Stream

The Network (`/network`) shows a live feed of everything your connections have posted — the social heart of your Hubzilla experience.

[IMAGE: Network page with filter bar and post list visible]

## View Modes

Use the view switcher (top-right of the filter bar, or bottom row on mobile) to change how posts are displayed:

| Mode | Description |
|------|-------------|
| **Feed** | Full posts in a vertical feed, one below the other |
| **Grid** (masonry) | Posts arranged in a Pinterest-style card grid |
| **List** | Compact single-line list — good for quick scanning |
| **Inbox** | Conversation-style view grouping threads together |

[IMAGE: View mode switcher with all four icons]

## Sort Order

Three buttons in the filter bar control sort order:

- **Latest** — newest posts first (by creation date)
- **Active** — posts with the most recent comment, like, or reaction appear first
- **All** — unthreaded flat list of every activity item in chronological order

## Quick Filter Toggles

These icon buttons are always visible in the filter bar. They can be combined freely.

| Button | What it shows |
|--------|--------------|
| ★ Star | Only your starred posts |
| 🔔 Following | Posts in threads you are following |
| 👤 Conversations | Threads where you have commented, liked, or reacted |
| ✉ DMs | Direct messages only |
| 📅 Events | Event posts only |

Click a toggle again to turn it off. An active toggle is highlighted in accent colour.

[IMAGE: Filter bar with a couple of toggles active]

## Filtering by Connection

1. Click the **person icon** (or the active connection chip if one is already selected).
2. Start typing a name — a suggestion list appears.
3. Select a connection to see only their posts.
4. Click the × on the chip to clear the filter.

[IMAGE: Connection filter with typeahead dropdown open]

## Search

Click the **magnifying glass** to open the search field.

- Type to search post text — results update as you type (400 ms delay).
- **Paste a URL** to import a post from a remote server — press Enter to fetch it.

[IMAGE: Search field expanded with URL pasted]

## Advanced Filters

Click the **filter icon** (≡ with lines) to open the advanced filter panel:

| Filter | Description |
|--------|-------------|
| **Tag** | Show only posts containing a specific hashtag |
| **From date** | Show posts created on or after this date |
| **To date** | Show posts created on or before this date |
| **Affinity min/max** | Filter by your affinity score with the author (0–100 scale) |

A small dot on the filter icon indicates that advanced filters are active.

[IMAGE: Advanced filter panel expanded showing all fields]

## Clear All Filters

When any filter is active, a **× button** appears at the right end of the filter bar. Click it to reset all filters and return to the default view.

## New Posts Banner

If new posts arrive while you are reading, a floating button appears at the top of the page:

> ↑ 3 new posts

Click it to load the new posts at the top of your stream.

## Load More

Scroll to the bottom — if there are more posts, a **Load more** button appears. Click it to fetch the next page. When you have seen everything, you'll see "You're all caught up".

[IMAGE: Load more button at the bottom of the stream]

## Refreshing

Click the **↺ refresh icon** at the left of the filter bar to reload the stream with the current filters.

## Stream Filters Widget

Sidebar version of the quick filter toggles, folders, tags, date range, and affinity slider described above — shown only to you (not visitors), so you can filter your Network stream without opening the panel on the page itself.

## Saved Searches Widget

Save your current combination of filters as a named search, then reapply it later with one click from this widget. Only visible to you.
