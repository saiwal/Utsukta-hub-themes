# Settings

Settings are at `/settings`. The left panel lists sections — click any section to open it on the right.

[IMAGE: Settings page with section list on left and content on right]

---

## Display

Controls the visual appearance of the interface.

[IMAGE: Display settings section]

| Setting | Description |
|---------|-------------|
| **Color scheme** | Choose a theme: Light, Dark, Nord, Dracula, Monokai, Gruvbox Dark, Catppuccin Mocha, Solarized Dark, Tokyo Night |
| **Custom colours** | Override individual theme colours (primary, background, etc.) |
| **Font size** | Small, Medium, Large, or XL |
| **Font family** | System default, Serif, Monospace, Nunito, Playfair, Comfortaa, Space Mono, Pacifico, Righteous, Comic, OpenDyslexic |
| **Thread mode** | How comment threads are displayed (flat or nested) |
| **Background image** | Set a custom background URL and choose Tile or Cover fitting |
| **Items per page** | How many posts to load per page |
| **Update interval** | How often the app checks for new content (seconds) |
| **Hubzilla theme** | The server-side Hubzilla theme (requires page reload) |

Click **Save** to apply changes. Theme and font changes apply immediately as you adjust them.

## Profile

Your public profile information — what others see when they visit your channel.

[IMAGE: Profile settings section]

| Field | Description |
|-------|-------------|
| **Display name** | Your visible name across the network |
| **Short description** | A one-liner about you (e.g. "developer, photographer") |
| **Homepage** | A URL shown on your profile |
| **Hometown** | Where you are from or based |
| **Gender** | Optional, shown on your profile |
| **Birthday** | Optional — month/day shown to connections |
| **About** | Longer bio or description |
| **Keywords** | Interests and tags used for directory discovery |
| **Hide friend list** | Toggle to hide your connections list from others |

## Account

Your login credentials and account-level settings.

[IMAGE: Account settings section]

- Change your **email address**
- Change your **password**
- Manage **two-factor authentication** (if enabled by the hub)

---

## Privacy

Control who can see your content and interact with you.

[IMAGE: Privacy settings section]

| Setting | Description |
|---------|-------------|
| **Permission limits** | Toggle advanced per-permission controls |
| **Default permissions** | Per-permission settings for each interaction type (who can view, comment, like, tag, etc.) |
| **Auto-permissions** | Automatically apply a permission set to new connections |
| **Opt out of directory indexing** | Remove yourself from the public directory |
| **Group actor** | Allow your channel to be used as a forum/community |
| **Allow all mentions** | Anyone can @mention you, not just connections |
| **Moderate unsolicited comments** | Hold comments from non-connections for approval |
| **Object capabilities (ocap)** | Enable capability-based access control |

## Notifications

Choose which events trigger emails and which show up in the visual notification widget.

[IMAGE: Notifications settings section]

### Auto-post Activity
Automatically share a post when you:
- Accept a friend request
- Join a forum or community
- Make an interesting profile change

### Email Notifications
Receive an email for:
- Connection request received
- Connection confirmed
- Someone posts on your wall
- New followup comment
- Private message received
- Friend suggestion
- Tagged in a post
- Poked or prodded
- Someone liked your post

### Visual Notifications
Show in the notification widget for:
- Unseen stream activity
- Unseen channel activity
- Unseen private messages
- Upcoming events / events today
- Upcoming birthdays
- System notifications and alerts
- Incoming items, forum posts, announcements, and more

## Danger Zone

Irreversible account actions. Proceed with care.

[IMAGE: Danger zone section with warning styling]

- **Export account data** — download all your content and connections
- **Delete channel** — permanently remove this channel and all its content
- **Delete account** — permanently remove your account, all channels, and all data

> These actions cannot be undone. The interface will ask you to confirm before proceeding.
